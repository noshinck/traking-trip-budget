🏔️ Hampta Pass Trek Tracker

The ultimate open-source financial companion for group trekking. Built to manage complex multi-payer expenses, emergency buffers, and real-time budget burn rate projections in high-altitude environments.

🚀 The Problem We Solved

Trekking with a group often leads to financial chaos: "Who paid for the train?", "Is the emergency fund touched?", "Are we overspending daily?".

Hampta Pass Trek Tracker brings professional financial planning to the trail. It keeps the group aligned, transparent, and financially safe.

✨ Key Features

Smart Pacing: Split tracking between transit (Delhi/Kozhikode) and high-altitude legs (Manali).

Emergency Lock: A visual, dedicated rescue pool that separates safety funds from spendable budgets.

Burn Rate Projection: Real-time forecasting to prevent running out of funds mid-trek.

Leaderboard Dynamics: Gamified contribution tracking with badges for high-contributors.

Offline-First Sync: Optimistic UI with automatic background sync to Supabase.

Memory Vault: Built-in cloud photo management for trek memories.

🛠️ Tech Stack

Frontend: React, TypeScript, Vite

Styling: Tailwind CSS, GSAP (for buttery smooth motion)

Backend: Supabase (PostgreSQL + Auth + Storage)

Deployment: Vercel

🚀 Getting Started

Clone the repo

git clone [https://github.com/noshinck/traking-trip-budget.git](https://github.com/noshinck/traking-trip-budget.git)


Install dependencies

npm install


Environment Setup
Create a .env file with your Supabase credentials:

VITE_SUPABASE_URL=your_url
VITE_SUPABASE_ANON_KEY=your_key


Run Development

npm run dev
